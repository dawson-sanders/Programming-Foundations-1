This program is similar to a excel sheet that performs a few math operations

PURPOSE:
- Be able to create a spreadsheet program that users can perform some math operations

TASKS:
- Store a specified value in the spreadsheet
- Store random values in the range low to high in the spreadsheet
- Calculate the minimum value in the spreadsheet
- Calculate the maximum value in the spreadsheet
- Calculate the sum of values in the spreadsheet
- Calculate the average value in the spreadsheet
- Calculate the standard deviation value in the spreadsheet
