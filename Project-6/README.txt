This program is similar to project 4 but is an object oriented version of the classic 1970's hit Colossal Cave Adventure 

AUTHOR:
- Dawson Sanders
- John Guach

PURPOSE:
- Be able to create an object oriented version of the Colossal Cave Adventure game

TASKS:
- Choose attributes to store in each room object
- Decide what operations to perform on each room object
- Create implementations of methods 

USER INSTRUCTIONS:
- Follow the command line interface prompts
