This program implements two variations of the classic caesar cipher.

AUTHOR:
- Dawson Sanders

PURPOSE:
- Encrypts/Decrypts user messages using the two versions of the caesar cipher

TASKS:
- Encrypt input messgaes by replacing each letter that occurs N characters to the left in the alphabet
- Decrypt messages by replacing each letter in the encoded message by the letter that occurs N characters to the right in the alphabet

USER INSTRUCTIONS:
- Follow the command line interface prompts
