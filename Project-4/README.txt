This program is a remake of the classic 1970's Colossal Cave Adventure computer game.

AUTHOR:
- Dawson Sanders
- John Guach

PURPOSE:
- The user gets to play a game where they travel through different rooms where they find gold/food and get into battles with a variety of creatures to eventually exit all the rooms

TASKS:
- Design a cave with 8 rooms
- Extend existing room functions
- Create new room functions
- Add room navigation
- Play the game

USER INSTRUCTIONS:
- Follow the command line interface prompts
